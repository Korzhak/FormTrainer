Щоб надіслати дані в форму User, Вам потрібно виконати наступні дії:

<form>
    <input type="text" placeholder="Your first name" name="first_name">
    <input type="text" placeholder="Your second name" name="second_name">
    <input type="text" placeholder="Your login" name="login">
    <input type="email" placeholder="youremail@mail.com" name="email">
    <input type="password" placeholder="Your password" name="password">
</form>

Замість [sitename] потрібно вкласти доменне ім'я сайту, або ІР-адрес 

<form action="[sitename]/form-trainer/user/">
    ...
</form>

Є два основних метода (використовуємо тільки їх):

  • POST
  • GET

Інші методи (використовуються рідше та в дуже вузькоспеціалізованих випадках):

  • PUT
  • DELETE
  • UPDATE

Вкладаємо назву методу на місце [method-name]

<form action="[sitename]/form-trainer/user/" method="[method-name]">
    ...
</form>

Є два основних метода (використовуємо тільки їх):

<form action="[sitename]/form-trainer/user/" method="[method-name]">
    ...
    <button type="submit">Відправити</button>
</form>
